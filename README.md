# Forensics

## Prerequisites

Make sure to have PNPM & Node installed, check `package.json` for allowed versions

## Quickstart

```bash
# 1. Install dependencies
pnpm install

# 2. Create your .env file
cp ~/.env.tpl ~/.env

# 3. Supply the project path to analyze in the .env file

# 4. Run the app
pnpm run dev
```
